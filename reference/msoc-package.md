# msoc

Package to make the `msoc` library
(`https://github.com/herumi/msoffice`) for encrypting/decrypting of
Office Open XML files available under R.

## Details

The password is expected to be plain text. If security is of importance,
do not use this package. If the password is lost, opening the file will
be impossible.
